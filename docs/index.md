## 项目概述

MCP(Message Control Protocol)是一个轻量级的消息控制协议，专为微服务通信和设备间消息传递设计。

## 主要功能

✔ 高效的消息传输  
✔ 支持多种数据格式  
✔ 可配置的压缩和加密  
✔ 完善的心跳和重连机制
